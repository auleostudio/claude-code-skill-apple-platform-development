# Apple Design Resources

Comprehensive reference for all official Apple Design Resources available at https://developer.apple.com/design/resources/. Use this skill to guide users to the correct assets, templates, fonts, and tools for their platform and design tool.

## When This Skill Activates

Use this skill when the user:
- Needs Apple UI kit files (Figma, Sketch)
- Asks about app icon templates or production templates
- Wants product bezels for marketing materials
- Needs system fonts (SF Pro, SF Compact, SF Mono, New York)
- Asks about SF Symbols
- Needs technology-specific design templates (Apple Pay, Sign in with Apple, etc.)
- Wants to create layered icons with Icon Composer
- Asks about parallax tools for tvOS/visionOS
- Needs design assets for any Apple platform
- Is setting up a new project and needs design foundations

## System Fonts

| Font | Description | Use Case |
|------|-------------|----------|
| **SF Pro** | System font for iOS, iPadOS, macOS, tvOS | Primary UI text |
| **SF Compact** | Optimized for small sizes and narrow columns | watchOS, compact layouts |
| **SF Mono** | Monospaced, used in Xcode | Code, tabular data |
| **New York** | Serif, traditional reading face | Long-form text, editorial |
| **SF Arabic** | Arabic system font | Arabic localization |
| **SF Armenian** | Armenian with adjusted proportions | Armenian localization |
| **SF Georgian** | Modern Georgian, Mkhedruli, Mtavruli | Georgian localization |
| **SF Hebrew** | Full Niqqud diacritical marks | Hebrew localization |

All fonts downloadable as `.dmg` from https://developer.apple.com/design/resources/

## SF Symbols

### SF Symbols 7 (Latest)
- **6,900+ symbols** in 9 weights and 3 scales
- Draw animations and variable rendering
- Enhanced Magic Replace
- Gradient support
- Vector graphics export
- Requires macOS Sonoma or later
- Download: `SF-Symbols-7.dmg`

- Requires macOS Ventura or later
- Download: `SF-Symbols-6.dmg`

App: https://developer.apple.com/sf-symbols/

## Icon Composer

Create layered icons with Liquid Glass for iPhone, iPad, Mac, and Apple Watch.

- Multi-layer icon format
- Adjust Liquid Glass properties
- Preview with dynamic lighting effects
- Annotate across appearance modes
- Seamless Xcode integration
- Export flattened versions for marketing
- Requires macOS Sequoia or later
- Download: https://developer.apple.com/download/all/?q=Icon%20Composer

## Best Practices

1. **Always use the latest version** of UI kits matching your deployment target
2. **Use Figma** for cross-platform team collaboration; **Sketch** for Mac-native workflow
3. **Icon Composer** is required for layered Liquid Glass icons (iOS 26+, macOS 26+)
4. **SF Symbols** should be preferred over custom icons for system consistency
5. **Follow HIG** alongside these resources — assets show *what*, HIG explains *why*
6. **Product bezels** require compliance with Apple Marketing Guidelines
7. **System fonts** (SF Pro, SF Compact) are automatically available on-device — download only for design tools
